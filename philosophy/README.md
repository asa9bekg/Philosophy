# Philosophy Hub

A modern, interactive website exploring the fascinating world of philosophy. This site features branches of philosophy, famous philosophers, inspirational quotes, thought experiments, a philosophy quiz, and contact functionality.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Dark Mode**: Toggle between light and dark themes
- **Search Functionality**: Search for philosophers, quotes, and sections
- **Interactive Elements**: Animated canvas background, quote generator, expandable sections
- **Philosophy Branches**: Explore Metaphysics, Epistemology, Ethics, Logic, Aesthetics, and Political Philosophy
- **Famous Philosophers**: Detailed profiles of Socrates, Plato, Aristotle, Descartes, Kant, and Nietzsche
- **Quote Generator**: Random philosophical quotes to inspire deep thinking
- **Thought Experiments**: Interactive explorations of classic philosophical dilemmas
- **Philosophy Quiz**: Test your knowledge with 10 challenging questions
- **Contact Form**: Get in touch with the philosophy community

## Technologies Used

- HTML5
- CSS3 (with animations, dark mode, and responsive design)
- JavaScript (ES6+)
- Font Awesome (for icons)
- Google Fonts (Playfair Display and Roboto)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Explore the various sections of the site

## Features in Detail

### Canvas Animation
The hero section features an animated particle system that visualizes the interconnected nature of philosophical ideas.

### Search Functionality
Use the search bar in the header to quickly navigate to different sections of the site.

### Dark Mode
Click the moon/sun icon in the header to toggle between light and dark themes. Your preference is saved locally.

### Quote Generator
Click the "Get Inspired" button or "New Quote" button to display random philosophical quotes.

### Branch Exploration
Click on any philosophy branch card to reveal detailed information about that area of study.

### Philosopher Profiles
Click "Learn More" on any philosopher card to see detailed biographical information.

### Thought Experiments
Explore classic philosophical thought experiments by clicking the "Explore" buttons.

### Philosophy Quiz
Test your knowledge with our 10-question philosophy quiz. Each question includes an explanation of the correct answer.

### Contact Form
Use the contact form to send messages to the philosophy community.

## Customization

- **Colors**: Modify the CSS custom properties in `styles.css` to change the color scheme
- **Content**: Update the quotes, philosopher profiles, quiz questions, and experiment details in `script.js`
- **Fonts**: Change the Google Fonts import in `index.html` for different typography
- **Quiz**: Add more questions to the `quizQuestions` array in `script.js`

## Troubleshooting

- **Canvas not displaying**: Ensure your browser supports HTML5 Canvas
- **Animations not working**: Check if CSS animations are enabled in your browser
- **Dark mode not persisting**: Ensure localStorage is enabled in your browser
- **Mobile menu not working**: Ensure JavaScript is enabled
- **Contact form not working**: This is a demo form - in production, you'd need a backend to handle submissions
